Cirqueti

A site for writers.

Cirqueti is a website designed to make it easy for writers to get their work online, build portfolios, and get critique.

Pieces of writing are added and revised through a simple, unobtrusive editing interface. Portfolios offer writers a simple “about the author” blurb, some contact information, and the ability to “showcase” their favorite authored items. The most important part of the system is an innovative javascript critiquing interface that makes it nearly effortless for anyone to pick a bit of text and comment on it.

Cirqueti's implementation is currently being finished. It will soon enter a small, private alpha testing phase before being opened up in a public beta.

This project is a collaboration with Rory Marinich, who planned and designed the interface; my task is developing the backend. Cirqueti is implemented in Python using the Django Web framework.

Light City

A Tumblr theme.

Light City is a theme for Tumblr, a popular mixed-media blogging platform. It was forked from “I Like Lampposts” to provide a much cleaner, simpler feel.

Since its acceptance into the theme garden (Tumblr's repository of user-designed themes), it has gained over 3,000 users.

A sample tumblelog using this theme is available here.

I Like Lampposts

A Tumblr theme.

I Like Lampposts is another Tumblr theme that I created, based on, well, lampposts. Using nearly the same clean HTML code as Light City, it provides an entirely different visual layout and feel.

This theme is unfinished and has therefore not been submitted to Tumblr's theme garden. It can be seen in use on a sample tumblelog in its partially completed state, however.

Not a Douche

A viral website.

Not a Douche describes itself as a sort of “antiFacebook”. Its stated idea is to “discourage interaction with other people”.

The interface is simple: just enter your name, then answer a series of questions you would probably never be asked by a real social networking site. When you've finished, you're taken to your “profile”, populated with your responses, and given a URL to hand out to those whom you think might be interested.

This was a quick, simple experiment to get my hands dirty in building Web apps, written in PHP. Within a few hours of its launch, however, the website had amassed several thousands of users—by no means other than word of mouth. It was also featured on several radio stations.

This project was a collaboration with Rory Marinich, who handled the site's design. You can find Not a Douche at notadouche.com. (Note: it is unmaintained, and changes in the database setup have rendered it mostly dysfunctional for the moment.)

LectureNotes

An application for students.

LectureNotes is a system designed to make electronic in-class note-taking both feasible for students and more effective than traditional paper-based methods.

The system consists of an efficient, rich editor and an XML vocabulary to describe notes. The interface aims to be extremely user-customizable and provide an effectively “lossless” storage format that maintains contextual information in addition to a simple notes tree.

This project was initiated through the Rensselaer Center for Open Source. Its (currently very incomplete) source and further information are available from its homepage.
